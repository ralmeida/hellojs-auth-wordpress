Hello.js Auth allows you to provide login and account creation in WordPress using Hello.js. 

== Description ==
The Hello.js allows your end user the ability to mostly seamlessly login using Facebook, Google or Github.

== Changelog ==
Initial version
- base functionality where in setup you can disable normal WP Login and Registration for Hello.js social accounts
- in WP Plugin Settings setup APPIDs for Facebook, Google, Github
