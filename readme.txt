HelloJs-Auth provides Social Auth in WordPress using the Hello.js library.

== Description ==
HelloJs-Auth is a plugin that will allow your end user the ability to seamlessly login using Facebook, Google, Github, or any of the oob providers supported by Hello.js.

The main focus of this project is not to build full Facebook or any other platforms social integration. Instead the aim is to provide a seamless easy way for your users to login and/or create an account on your WordPress site.

Requires: PHP 5 >= 5.2.0 (for json_encode)
